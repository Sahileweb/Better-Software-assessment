# Upvote - Feature Request Board

A clean, minimalist product feedback board built for the Better Software assessment.

## Tech Stack
* **Frontend:** React, Vite, Tailwind CSS
* **Backend:** Python, Flask, SQLAlchemy, SQLite

## How to Run Locally

### 1. Backend Setup
\`\`\`bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate
pip install -r requirements.txt
python app.py
\`\`\`

### 2. Frontend Setup
Open a new terminal:
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`
